# Finance Projects Portfolio

This is a GitHub Pages portfolio with a homepage and one detail page per finance project.

## File structure

Upload these files and folders to the root of your `your-username.github.io` repository:

```text
index.html
style.css
projects/
  monte-carlo-option-pricer/
    index.html
  correlation-cointegration/
    index.html
  market-dashboard/
    index.html
```

## What to edit first

Replace these placeholders in all HTML files:

- `Your Name`
- `your.email@example.com`
- `your-username`
- `https://your-option-pricer-demo.streamlit.app`
- `https://your-cointegration-demo.streamlit.app`
- `https://your-market-dashboard.streamlit.app`

## How it works

The homepage links to:

```text
/projects/monte-carlo-option-pricer/
/projects/correlation-cointegration/
/projects/market-dashboard/
```

Each project page includes:

- project objective
- finance method
- key outputs
- short code excerpt
- `View full code` button
- `Live demo` button

## Best recruiter flow

Use this structure on your CV:

```text
Portfolio: https://your-username.github.io
```

Then each project page should send the recruiter to:

1. the GitHub repository for the full code
2. the Streamlit link for the live demo
