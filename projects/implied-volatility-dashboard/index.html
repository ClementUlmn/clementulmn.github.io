Project 02

Correlation & Cointegration

Cross-asset research to identify relationships, mean-reversion opportunities, and trading signals.

View full code Live demo

Finance objective

Test whether two assets share a stable long-term relationship and use the spread deviation to generate potential statistical arbitrage signals.

Input Asset prices and returns
Method Rolling correlation, cointegration, z-score
Output Pairs, spreads, entry signals

What it shows

  • Market data analysis
  • Statistical testing
  • Trading signal design
  • Research mindset for markets

Methodology

  • Download and clean historical price series.
  • Compute rolling correlations to detect changing relationships.
  • Run cointegration tests on selected asset pairs.
  • Build a z-score signal from the spread.

Key code excerpt

spread = asset_a - hedge_ratio * asset_b
z_score = (spread - spread.rolling(60).mean()) / spread.rolling(60).std()

signals = pd.Series(index=spread.index, dtype="float")
signals[z_score < -2] = 1
signals[z_score > 2] = -1
