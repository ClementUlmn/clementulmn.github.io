Finance objective

Estimate the fair value of a European call or put by simulating terminal prices under a risk-neutral process, then compare the result with the closed-form Black-Scholes price.

Input Spot, strike, volatility, rate, maturity

What it shows

  • Derivatives knowledge
  • Simulation and model validation
  • Clean Python implementation
  • Clear link between math and markets

Methodology

  • Simulate terminal asset prices under geometric Brownian motion.
  • Compute discounted option payoffs under the risk-neutral measure.
  • Benchmark the Monte Carlo estimate against Black-Scholes.
  • Estimate Greeks to understand sensitivity to market inputs.

Key code excerpt

def monte_carlo_call(S0, K, r, sigma, T, n_paths):
    z = np.random.standard_normal(n_paths)
    ST = S0 * np.exp((r - 0.5 * sigma**2) * T + sigma * np.sqrt(T) * z)
    payoff = np.maximum(ST - K, 0)
    return np.exp(-r * T) * payoff.mean()
